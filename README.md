    Pac-Man

Este projeto é uma recriação do clássico **Pac-Man**, desenvolvido em **C#** utilizando a **Uno Platform**.

O objetivo é reproduzir a experiência original do jogo de forma academica aplicando conceitos, como:

- Programação Orientada a Objetos (POO)
- Lógica e loop de jogo
- Movimentação baseada em grid
- Sistema de colisões
- Gerenciamento de estados (jogador, fantasmas, power mode)

---

O projeto está organizado da seguinte forma:

- `apresentacao/` — Materiais visuais, imagens e GIFs demonstrativos  
- `codigo-fonte/` — Implementação completa do jogo  
- `documentacao/` — Documentação técnica
