# 1 - Proposta do Jogo / Introdução

Este projeto propõe a recriação do clássico **Pac-Man**, desenvolvido em **C#** com a **Uno Platform**.

O objetivo principal é aplicar, na prática, conceitos fundamentais de desenvolvimento de software no contexto de jogos digitais, recriando as principais mecânicas do jogo original, como movimentação em grid, sistema de colisões e interação com fantasmas.

Durante o desenvolvimento, foram aplicados conceitos como:

- Programação Orientada a Objetos (POO)  
- Separação de responsabilidades  
- Organização e arquitetura de código  
- Gerenciamento de estados (modo normal e power mode)  
- Uso de eventos para desacoplamento da lógica
